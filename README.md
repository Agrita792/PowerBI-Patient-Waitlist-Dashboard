# PowerBI-Patient-Waitlist-Dashboard
Developed an interactive Power BI dashboard analyzing 700K+ patient wait-list records across specialties, age groups, and case types. Performed data cleaning, transformation, and modeling, created DAX measures using CALCULATE, SUM, AVERAGE, MEDIAN, SWITCH, and IF, and delivered actionable insights including an 11% YoY increase in wait-list volumes and identification of high-delay specialties for operational decision-making.
